# Trendy Stickers - Full Stack E-commerce Application

A full-stack e-commerce web application built using React, Spring Boot, and H2 Database.

## Technologies Used

### Frontend
- React
- React Router
- Axios
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 Database

## Features
- User Authentication
- Product Listing
- Shopping Cart
- Contact Form
- Responsive UI
- REST APIs
- JWT Authentication
- Order Management

## Project Structure

eazystore-ui/   -> Frontend  
eazystore/      -> Backend

### Frontend
